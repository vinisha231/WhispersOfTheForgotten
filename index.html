Whispers of the Forgotten

By Team F15

Description

Our game is a 2D horror escape room where the player must find keys and run from a kidnapper to escape. Inspired by games like "Little Nightmares" and "Among the Sleep." Unique features include a barking NPC dog companion and dynamic kidnapper AI.
